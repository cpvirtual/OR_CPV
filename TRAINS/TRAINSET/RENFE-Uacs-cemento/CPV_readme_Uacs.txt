========== Instruções - PT ==========
Para ter estes vagões disponíveis na Portugal79 ou outros cenários da CPVirtual, por favor faça o seguinte:

- Descarregue os vagoes "Vagones tolva para cemento Uacs de TRANSFESA" em http://www.trensim.com/lib/msts/index.php?act=view&id=419
- Instale os vagoes na pasta onde descarrega o repositório da CPV. ( Na pasta que contém a subpasta Global )

Depois de feito já poderá usufruir destes vagões nas composições fornecidas no repositório da CPV



========== Instruction - EN ==========
In order to have these wagons available for use with Portugal79 and other CPv routes, you will have to download them and add them to the repository copy you have on your hard drive. Please do the following:

- Download "Vagones tolva para cemento Uacs de TRANSFESA" at http://www.trensim.com/lib/msts/index.php?act=view&id=419
- Install the contents of the exe in the folder were you keep the repository. (It should have a subfolder named Global )

After that you will have these wagons available on the consists we provide in the repository.
